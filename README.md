# 🖥️ Python Mini Portal (Tkinter GUI)

A simple GUI-based **Mini Portal** developed using **Python** and **Tkinter** as my **first internship task**. This project demonstrates the basics of GUI development, event handling, and Python programming through a clean and interactive interface.

## 📌 Features

- 🏠 Home Page
- ➕ Simple Calculator (Addition)
- 📝 Quick Notes Section
- 📋 Sidebar Navigation
- 💬 Message Box Notifications
- 🎨 User-Friendly GUI

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI Library)

## 📚 Concepts Used

- Functions
- Tkinter Widgets
- Frames
- Buttons
- Labels
- Entry Widgets
- Text Widget
- Message Box
- Event Handling
- Layout Management (`pack`)

Mini-Portal/

## ▶️ How to Run

1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Make sure Python 3 is installed.
4. Run the project:

```bash
python main.py
```

## 🚀 Application Modules

### 🏠 Home
Displays a welcome message for the user.

### ➕ Calculator
Allows users to enter two numbers and calculate their sum.

### 📝 Notes
Provides a simple notepad where users can write notes and receive a confirmation message after saving.
